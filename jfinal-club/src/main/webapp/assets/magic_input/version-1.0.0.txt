文档：
1：设置input大小的class
   sm 是 small的缩写 , lg 是 large缩写

   在 Checkbox中设置下面class
    mgc-sm mgc-lg

   在 Radio Button中设置下面
    mgr-sm mgr-lg

2：改变颜色的 Class
   在 Checkbox中设置下面class
    mgc-primary mgc-info mgc-success mgc-warning mgc-danger

   在 Radio Button中设置下面
    mgr-primary mgr-info mgr-success mgr-warning mgr-danger
